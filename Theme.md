3DGS RT

Одно из новых популярных направлений в компьютерной графике является создание и визуализация сцен с помощью 3D Gaussian Splatting (3DGS) из набора изображений. Такой подход довольно эффективен в сравнении с нейронными сетями за счет высокой скорости визуализации, поскольку используется простая растеризация.

Однако при таком подходе не используется какая либо модель освещения, что не позволяет изменять освещенность сцены, строить тени с правильной геометрий для различных объектов, отображать корректные отражения и преломления, например, для стеклянных объектов, и многие другие эффекты.

Добавление модели освещения, основанной на Physically Based Rendering (PBR) техниках и метода Ray Tracing (RT), позволит этого достичь, что потребует создания дополнительных параметров и исследования их влияния на сцену.

В рамках работы предполагается, что студент должен посмотреть облако гаусианнов с помощью готовых инструментов, дополнительная сложность заключается в том, что потребуется добавить новые свойства (параметры) для гауссианов и источники света с целью моделирования различных эффектов.

Для достижения поставленных целей необходимо будет изучить, описать и реализовать модель освещения, основанной на PBR техниках, изучить влияние и обосновать использование новых параметров для гауссианов. Ожидается, что будет построены приложение и выполнено качественное и количественное сравнение его работы по сравнению с первоначальным вариантом.
